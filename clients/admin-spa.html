Admin client

The Cornflow admin app is an SPA app that handles the basic functionality of the REST API to administer a Cornflow Server. It can also be used by any user to communicate with the server.

It allows the:

  1. Management of users.

  2. Management of instances and their executions.

  3. Creation of new instances.

  4. Creation of new executions.

  5. Edition or deletion of instances and executions.

  6. Visualization of execution progress and statistics.

The app is built in VueJS.

An example instance running can be found in:

https://devsm.cornflow-admin.baobabsoluciones.app/
